Это мой первый pet-проект на GO!

Это обычный CLI-трекер для создания и отслеживания задач. 

Установка простая - устанавливаем данный проект к себе на компьютер, и в терминале переходим в директорию с проектом. 
В терминале мы вводим go run ./main.go "command". Для просмотра списка команд вводим: 

go run ./main.go help

Приятного использования!
